# Pomodoro Focus App 🍅

A productivity timer based on the Pomodoro technique.

## Features
- Work and break sessions
- Countdown timer
- Reset functionality
- Visual progress tracking

## Concepts Used
- Python
- Tkinter
- Canvas
- Timer Events
